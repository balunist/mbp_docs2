Grain Orientation on Boards

By default the component’s grain is determined by it’s longest edge. This can be overridden by applying a 3D appearance and enabling Automatic Grain Alignment. The Component grain determined by either method can be forced to rotated 90 degrees by choosing Rotate when a component has been selected as shown below. See Selection Support for details.


The component’s grain will be aligned with the grain on the targeted board. Grain direction on the targeted board is selectable by toggling the double-arrowed icon horizontal (X) or vertical (Y) for board types on the lumber tab or when optionally entering stock lumber.
